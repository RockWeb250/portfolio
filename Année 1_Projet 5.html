Projet 5 - Base De Données : Surveillance des Mesures Environnementales

← Retour aux projets

Résumé du Projet

Création d'une base de données pour surveiller les mesures environnementales dans plusieurs villes, permettant une analyse rapide et facilitée des données acquises par capteurs.

Compétences Acquises

  • Création et gestion de bases de données relationnelles
  • Traitement et analyse de données environnementales
  • Optimisation des requêtes pour une recherche rapide

Logiciels Utilisés

  • MySQL pour la gestion de base de données
  • Python pour le traitement et l'analyse des données

Utilité dans le Cadre Professionnel

Ce projet a permis de développer des compétences en analyse de données et en gestion d'information, essentielles pour des postes dans l'analyse environnementale et la science des données.
