← Retour aux projets

Résumé du Projet

Réalisation d'un site de recherche de stages simple d'utilisation pour étudiants où leurs tuteurs pourraient suivre leurs candidatures.

Compétences Acquises

  • Utilisation de l'architecture MVC dans le cadre du développement web
  • Utilisation du FrontEnd et Backend.
  • Utilisation de requêtes SQL.
  • Sécurité d'un site (empêcher failles XSS).
  • Méthodologie agile (SCRUM).

Technologies Utilisées

  • Langages HTML, CSS, JavaScript, PHP et Twig.
  • Base de données MySQL pour la gestion des données.
  • GitHub pour coder en équipe.
  • PHPmyAdmin pour les Tests Unitaires.
  • Jira pour la méthodologie Scrum.

Utilité dans le Cadre Professionnel

Ce projet m'a permis de développer des compétences en développement web, essentielles pour concevoir et sécuriser des sites modernes adaptées aux besoins des utilisateurs.
